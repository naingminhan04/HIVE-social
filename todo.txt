### Chat View UI Improvements

#### Message Grouping

* When consecutive messages are sent by the same user, group them together instead of treating each message as a separate chat item.
* Display the user's avatar/profile picture only once per message group, similar to Messenger, iMessage, Telegram, or WhatsApp.
* Position the avatar at the bottom of the grouped messages rather than repeating it beside every message bubble.

#### User Names

* In 1-to-1 conversations:

  * Do not display sender names above messages.
* In group conversations:

  * Display the sender's name only once at the beginning of a message group.
  * Use a smaller, muted text style.
  * Place the name outside the chat bubble for a cleaner appearance.

#### Timestamps

* Reduce timestamp clutter by:

  * Showing timestamps only when there is a meaningful gap between messages.
  * Alternatively, display a timestamp for the entire message group instead of every individual message.
* Use a subtle, low-contrast style to keep focus on the conversation.

#### Expected Result

The chat interface should feel cleaner, more modern, and more space-efficient by minimizing repeated avatars, names, and timestamps while maintaining clear message ownership and readability.
